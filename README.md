# Anime Platform

A full-stack anime web app with:

- User authentication (JWT)
- Watchlist feature
- Anime search & genre filtering
- Personalized recommendations

## Tech Stack

Frontend:
- React (Vite)
- Axios

Backend:
- Node.js
- Express
- MongoDB

## Features

- Login & Register
- Add/Remove Watchlist
- Like Anime
- Genre-based browsing
- Recommendations

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
